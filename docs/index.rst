Pynomaly Documentation
=====================

Welcome to Pynomaly's documentation!

Pynomaly is a Python anomaly detection package targeting Python 3.11+ with clean architecture principles, integrating multiple ML libraries (PyOD, PyGOD, scikit-learn, PyTorch, TensorFlow, JAX) through a unified interface.
