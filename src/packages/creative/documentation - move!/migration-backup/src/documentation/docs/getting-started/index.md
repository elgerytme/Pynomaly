## 🔧 System Requirements

### **Minimum Requirements**
- **Python**: 3.8+
- **Memory**: 4GB RAM
- **Storage**: 2GB available space

### **Recommended Requirements**
- **Python**: 3.11+
- **Memory**: 8GB RAM
- **Storage**: 5GB available space
- **GPU**: CUDA-compatible (for deep learning models)
