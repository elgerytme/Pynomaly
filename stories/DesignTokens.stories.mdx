## =' Usage Guidelines

### Color Usage

- **Primary Blue**: Use for primary actions, links, and brand elements
- **Success Green**: Indicate successful operations and normal states
- **Warning Orange**: Show warnings and potential issues
- **Danger Red**: Highlight errors and anomalies
- **Neutral Grays**: For text, backgrounds, and subtle UI elements

### Typography Hierarchy

1. Use heading sizes to create clear hierarchy
2. Maintain consistent line heights for readability
3. Use font weights to create emphasis
4. Prefer Inter for UI text, JetBrains Mono for code/data

### Spacing Consistency

- Use the 4px base grid for all spacing decisions
- Maintain consistent gaps between related elements
- Use larger spacing to separate distinct sections
- Follow the spacing scale rather than arbitrary values

### Accessibility Considerations

- Ensure sufficient color contrast (4.5:1 for normal text, 3:1 for large text)
- Provide clear focus indicators for keyboard navigation
- Use semantic color meanings consistently
- Test with screen readers and assistive technologies
